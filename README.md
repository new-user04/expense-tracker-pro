# Expense Tracker Pro

A full-stack expense management application built using Spring Boot, MySQL, Thymeleaf, and Spring Security.

## Features
- User Registration & Login
- BCrypt Password Encryption
- Add/Edit/Delete Expenses
- User-specific Expense Tracking
- Analytics Dashboard
- CSV Export
- Responsive Bootstrap UI

## Tech Stack
- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- Thymeleaf
- Bootstrap
